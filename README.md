# The Literary Ledger

**The Literary Ledger** is a full-stack web application designed for book enthusiasts to discover, track, and review their favorite reads. By leveraging the **Open Library API**, users can access a vast database of real-world book data and manage their own personalized digital bookshelves.

---

## 🚀 Features

- **Book Discovery**: Browse featured or popular books and search for specific titles, authors, or keywords.
- **Detailed Information**: Access summaries, book covers, and metadata for any book in the database.
- **Personalized Bookshelves**: Organize your reading life by marking books as **Read**, **Currently Reading**, or **To Be Read (TBR)**.
- **User Reviews**: Write, edit, and delete reviews for books you’ve read.
- **User Profiles**: A dedicated space to view your bio, profile picture, submitted reviews, and categorized bookshelves.
- **Secure Authentication**: Log in or register using email/password or Google Sign-In via Firebase Authentication.

---

## 🛠️ Tech Stack

### Frontend

- **React (JavaScript)**: For building a component-based user interface.
- **Tailwind CSS**: For modern, responsive styling and UI design.
- **React Router**: For seamless navigation between application pages.
- **React Context**: Used for global authentication state management.

### Backend & Services

- **Firebase Authentication**: Manages user accounts and secures protected routes.
- **Cloud Firestore**: A NoSQL database used to store persistent data for users, reviews, and bookshelf entries.
- **Open Library API**: The primary source for fetching real-life book data.
